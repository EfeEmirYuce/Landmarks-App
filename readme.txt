This is a small project about marking locations on a map for tracking visits, planing future plans, adding notes for visits.

Built with Node.js, Express, MySQL, HTML, CSS, Javascript

Dependincies are body-parser, cors, express, mysql2 and can be checked in package.json


SETUP:

git clone https://github.com/your-username/landmark-management.git
cd landmark-management

npm install

node server.js
